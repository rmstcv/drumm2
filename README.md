# **Drumm2**
### **Drumm2** is an online step sequencer for quickly creating electronic music from samples
<br>
<a href="https://rmstcv.github.io/drumm2/build"><img src="drumm2.jpg" width="400"></a>
<br>

___

### Application features:
* Choose samples
* Change BPM
* Play solo track
* Mute track
